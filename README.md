# Web Scraping Data From YouTube
Download the `.otd` file, import into Octoparse, a free web scraping App, and get the web data!
## Scraping Video Information from YouTube<br>
### Data extracted: <br>
 * Title<br> 
 * Duration<br>
 * Publisher<br>
 * Published Time<br>
 * Views<br>
 * Likes and Dislikes<br><br>
