# Web-Scraping on YouTube
Download the `.otd` file, import into Octoparse, `a free web scraping App,` and get the web data
## Scraping Video Information on YouTube<br>
### Data extracted: <br>
 * Title<br> 
 * Duration<br>
 * Publisher<br>
 * Published Time<br>
 * Views<br>
 * Likes and Dislikes<br>
